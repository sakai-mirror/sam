Using the WYSIWYG editor

In certain tools, you can control the graphical appearance of your text using the WYSIWYG editor. The WYSIWYG toolbar has icons for editing and styling your text. Also, you can copy and paste from Microsoft Word files and retain most formatting.

The WYSIWYG editor is based on an open-source application called FCKeditor. To see what each icon in the toolbar does, refer to the Overview page in the FCKeditor Users Guide at:

http://wiki.fckeditor.net/UsersGuide

Browser considerations

To ensure that the WYSIWYG editor appears and functions properly, you must use a supported browser; see Which browser should I use?

However, Internet Explorer is currently the only browser that supports the paste function within the editor. If you're using Firefox, see Problems pasting text into FCKeditor in Firefox.

Also, Internet Explorer is currently the only browser that supports the spellcheck feature. To use spellcheck, you must first download and install the ieSpell extension from the ieSpell download page at:

http://www.iespell.com/download.php
