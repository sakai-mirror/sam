Differences between Create and Quick Create in Tests & Quizzes

When creating an assessment, the default settings are different when you click Create from the defaults for Quick Create, unless overridden by the default settings of a particular Assessment Type. Each Assessment Type may also have some variation in settings depending on whether you click Create or Quick Create; see below.
