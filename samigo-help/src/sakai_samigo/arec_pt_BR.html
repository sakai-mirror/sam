Ordenação de perguntas

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

Na página de Avaliações, clique no título da avaliação que você deseja trabalhar. Você verá uma lista de perguntas para esta avaliação. Para cada pergunta, na lista drop-down "Question", selecione a ordem numérica na qual você deseja que a pergunta apareça. Por exemplo, se você deseja que a pergunta seja a terceira pergunta, selecione 3.

Quando você reordenar uma pergunta, moverá perguntas que estavam originalmente acima dela para baixo na lista, e pergunts que estavam originalmente abaixo dela para cima na lista. Por exemplo, se você usar a lista drop-down para mudar a pergunta 3 para a pergunta 1, então a pergunta 1 se tornará a nova pergunta 2, e a pergunta 2 se tornará a nova pergunta 3. Se você usar a listat drop-down para mudar a pergunta 3 para a pergunta 5, então a pergunta 4 se tornará a nova pergunta 3, e a pergunta 5 se tornará a nova pergunta 4.

Nota: Você não pode configurar a ordem das perguntas se a parte que as contém está configurada par ordem aleatória.
