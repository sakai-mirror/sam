Criando perguntas para uma avaliação

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

A criação de perguntas é somente um dos passos envolvidos na criação de uma avaliação; veja também:

  • Criando uma nova avaliação
  • Configurações da avaliação
  • Adicionando, movendo, copiando, ou removendo um conjunto ou sub-conjunto de perguntas

Você pode escolher entre vários tipos de perguntas durante a construção de uma avaliação; veja Tipos de perguntas disponíveis para avaliação.

  1. No menu do curso ou site de projeto, clique em Testes & Quizzes. Para criar uma nova avaliação, sob o cabeçalho "Nova Avaliação", digite o nome para a sua avaliação no campo "Título".

    Você pode usar o menu drop-down ao lado de Escolha um tipo de Avaliação existente para escolher um template para sua avaliação. Você pode importar também uma avaliação usando a opção Importar. Veja Importando ou exportando uma avaliação.

    Quando tiver feito suas escolhas, você tem duas opções para criar a avaliação:

    • A opção Criar Rápido lhe permite gerar perguntas para sua avaliação, digitando-as em um formato particular ou copiando e colando-a de um arquivo de texto formatado apropriadamente. Instruções de formatação estão disponíveis no rodapé da página abaixo de Exemplos & de Instruções.

    • A opção Criar leva ao editor de perguntas. Para começar a adicionar perguntas para uma nova avaliação, escolha o tipo de pergunta que você quer adicionar do menu drop-downAdicionar Pergunta:. Abaixo estão as instruções para cada tipo de pergunta.

      Nota: As configurações padrão para sua avaliação será diferente dependendo de qual opção você escolhe Criar ou Criar Rápido; veja Diferenças entre Criar e Criar Rápido nos Testes & Quizzes.

    Múltipla Escolha

    Para criar uma Pergunta Múltipla Escolha:

    1. Na tela de Perguntas no menu drop-downAdicionar Pergunta:, selecione Múltipla Escolha. Leva para a tela de edição.

    2. No campo "Valor do Ponto da Resposta", entre com o valor do ponto que você quer atribuir para esta pergunta. Entre 0 (zero) para uma pergunta sem nota.

    3. No campo "Texto da Pergunta", entre com o texto para a pergunta. Clique Exibir/Ocultar Editor Rich-Text para mostrar a barra de ferramentas WYSIWYG.

    4. Para adicionar um anexo, clique Adicionar Anexos e entre com um nome ou procure um arquivo ou entre com uma URL.

    5. Na seção "Resposta", escolha entre os seguinte:

      • Única correta (por padrão)
      • Múltiplas Corretas, Seleção Simples
      • Múltiplas Corretas, Seleção Múltipla

      Conforme você muda a seleção, a tela irá atualizar para habilitar ou desabilitar os campos de requisitos.

    6. No caso de uma resposta única correta, você verá quatro caixas de texto para possíveis respostas. Para indicar qual das respostas é a correta, clique no botão de seleção para isso. No caso de múltiplas respostas corretas, cheque nas caixas ao lado das respostas corretas.

      Para apresentar poucas possíveis respostas, clique em Remover abaixo das caixas que você quer remover. Para apresentar mais possibilidades, use o menu drop-down Inserir respostas adicionais para selecionar o número de possibilidades que você deseja adicionar.

    7. Se você queiser que as possíveis respostas apareçam em uma ordem aleatória sob "Personalizar respostas", clique em Sim. Para que apareçam em ordem (por padrão), veja na tela de edição de perguntas, clique em Não.

    8. Sob "Requer Justificativa", escolha Sim ou Não (padrão) dependendo se você quer que o usuário forneça uma justificativa para a escolha de sua resposta.

    9. A partir de Atribuir por parte no menu drop-down, escolha a parte (ou seja, seção) para a qual você quer atribuir esta pergunta. Deixe-o como valor padrão se você não criou nenhuma parte.

    10. Para atribuir esta pergunta a um conjunto de perguntas, a partir de Atribuir a um conjunto de perguntas no menu drop-down, selecione o nome do conjunto desejado.

    11. Sob "feedback de resposta Correta/Incorreta", você pode fornecer um feedback de nível de pertunta para respostas corretas e/ou incorretas.

    12. Quando terminar, clique em Salvar para voltar à tela de edição de avaliação. Você verá que as suas novas perguntas foram adicionadas na avaliação.

    Survey

    O processo de criação de perguntas de pesquisa é semelhante ao usado para criar perguntas de múltipa escolha, exceto pelas seguintes diferenças:

    • O padrão do campo "Valor do Ponto de Resposta" é 0 (zero).

    • Para o texto de possíveis respostas, você pode escolher de um conjunto de escalas pré-definidas coumumente utilizada para relatórios de pesquisa. Para selecionar a escala que você quer usar, clique na opção de seleção correspondente. As escolhas são:

      • Sim, Não
      • Concordo, Discordo
      • Concordo, Indeciso, Discordo
      • Abaixo da Média -> Acima da Média
      • Concordo Plenamente -> Discordo Plenamente
      • Excelente -> Inaceitável
      • 1 -> 5
      • 1 -> 10

    • Você não tem a opção para personalizar respostas ou solicitar uma justificativa

    Resposta Curta/Resumo

    Para criar uma resposta curta/pergunta resumida:

    1. Na tela de Perguntas, em Adicionar Pergunta: no menu drop-down, selecioneResposta Cura/Resumo.

    2. No campo "Valor do Ponto da Resposta", entre com o valor do ponto que você quer atribuir para esta pergunta. Entre com 0 (zero) para uma pergunta sem nota.

    3. No campo "Texto da Pergunta", entre com o texto para a pergunta. Clique em Exibir/Ocultar o Editor Rich-Text para mostrar a barra de ferramentas WYSIWYG.

    4. Para inserir um anexo, clique em Adicionar Anexos e entre com um nome ou procure um arquivo ou entre com uma URL.

    5. A partir de Atribuir por parte no menu drop-down, escolha a parte (ou seja, seção) para a qual você deseja atribuir esta pergunta. Deixe-o com o valor padrão se você não criou nenhum parte.

    6. Para atribuir esta pergunta a um conjunto de perguntas, a partir de Atribuir a um conjunto de perguntas menu drop-down, selecione o nome do conjunto desejado.

    7. Na seção "Resposta:", você pode optar por fornecer um modelo de respostas curta ou um feedback de nível de pergunta digitando nos campos fornecidos.

    8. Quando terminar, clique em Salvar para voltar à tela de edição de avaliação. Você verá que as suas novas perguntas foram adicionadas na avaliação.

    Preencha a Lacuna

    Para criar uma pergunta do tipo preencher lacuna:

    1. Na tela de Perguntas, em Adicionar Pergunta no menu drop-down, selecione Preencha a Lacuna.

    2. No campo "Valor do Ponto da Pergunta" entre com o valor que você quer atribuir para esta pergunta. Entre com 0 (zero) para uma pergunta sem nota.

      Nota: O total de pontos que você atribuir será divido pelo total de todas as lacunas da pergunta. Por exemplo, pegue a seguinte pergunta:

      Polonius Laertes informa: "Não seja um {devedor} nem um {credor}.

      Se você atribuir 10 pontos para esta pergunta, os alunos obterão 5 pontos para cada lacuna "devedor" e "credor".

    3. No campo "Texto da Pergunta", entre com o texto para a pergunta Clique em Exibir/Ocultar Editor Rich-Text para mostrar a barra de ferramentas WYSIWYG. Coloque entre chaves ( { } ) os textos que indicam uma lacuna, por exemplo: Em Unix, para procurar o diretório corrente, digite {pwd} no prompt.

      Outras opções para a construção de perguntas do tipo Preencha a Lacuna incluem:

      • Curingas: Use um asterisco ( * ) para representar um ou mais caracteres numa resposta. response. Por exemplo: Um adverbio, como {*mente}, modifica um verbo.
      • Também|Ou: Se duas ou mais respostas específicas puderem pertencer pertencer a uma lacuna, você pode usar um pipe ( | ) para separá-los. Por exemplo: {vermelho|amarelo|azul} é a cor primária que eu mais gosto.
      • Maiúsculas e minúsculas: Marque Maiúsculas e minúsculas caso a resposta exija. Por exemplo: A capital da Califórnia é {Sacramento}.
      • Exclusividade Mútua: Marque Mutuamente Exclusiva se sua pergunta tem duas ou mais lacunas com escolha de respostas idênticas, mas o aluno deve responder cada uma diferentemente. Por exemplo: Quando você lança uma moeda, ela cai em {cara|coroa} ou {cara|coroa}.

        Se as respostas são mutuamente exclusivas, o aluno deve digitar "cara" em uma lacuna e "coroa" na outra lacuna para responder à pergunta corretamente. Se as respostas não foram mutuamente exclusiva, então digite "cara" em ambas as lacunas e será considerado correto.

        Você pode usar a opção mutuamente exclusivo somente quando é aplicadoa todas as lacunas na pergunta. Por exemplo, pegue a seguuinte questão:

        Misturamos {vermelho|azul} e {vermelho|azul} para fazer {roxo}.

        Se você selecionar a opção mutuamente exclusiva para a questão acima, ela automaticamente se desmarcará quando você salvar a sua pergunta.

    4. Para adicionar um anexo, clique Adicionar Anexos e entre com um nome ou procure um arquivo ou entre com uma URL.

    5. Se sua avaliação tiver mais de uma parte (ou seja, seção), use Atribuir por parte no menu drop-down para atribuir a pergunta a uma parte específica.

    6. Para salvar a pergunta em um conjunto de perguntas para uso posterior, em Atribuir a um conjundo de perguntas no menu drop-down menu, selecione o nome do conjunto desejado. Para isso, você deve ter criado pelo menos um conjunto de perguntas; veja Adicionando, movendo, copiando, ou removendo um conjunto ou um sub-conjunto de perguntas.

    7. Os campos de "feedBack de resposta Correta/Incorreta" permitem que você defina mensagens opcionais para alunos. Use as configurações de avaliações de feedback para determinar se e quando os alunos veem estas mensagens. Veja. Configurações da Avaliação.

    8. Quando terminar, clique em Salvar para voltar à tela de Perguntas. Você verá quer suas novas perguntas foram adicionadas na avaliação.

    Resposta Numérica

    Para criar uma pergunta de resposta numérica:

    1. Na tela de Perguntas em Adicionar Pergunta: no menu drop-down, selecione Resposta Numérica.

    2. No campo "Valor do Ponto da Resposta", entre com o valor do ponto que você quer atribuir para essa pergunta. Entre com 0 (zero) para uma pergunta sem nota.

    3. No campo "Texto da Pergunta", entre com o texto para a pergunta. Clique em Exibir/Ocultar Editor Rich-Text para mostrar a barra de ferramentas WYSIWYG. Coloque entre chaves ( { } ) os caracteres que correspondem à lacuna, por exemplo: 4*4={16}

      Insira um pipe ( | ) entre os valores para delimiaá-los, por exemplo:

      A idade dos três é {121|145} anos.

      Todos os valores entre 121 e 145 serão respostas válidas.

    4. Para adicionar um anexo, clique em Adicionar Anexos e entre com um nome ou procure um arquivo ou entre com uma URL.

    5. No menu drop-down Atribuir por parte, escolha a parte (ou seja, seção) para a qual você quer atribuir esta pergunta. Deixe-o com o valor padrão se você não criou nenhuma parte.

    6. Para atribuir uma pergunta a um conjunto de perguntas, no menu drop-downs Atribuir a um conjunto de perguntas, selecione o nome do conjunto desejado.

    7. Sob "feedback de resposta Correta/Incorreta, você pode escolher para fornecer um feedback de nível de pergunta.

    8. Clique em Salvar para voltar à tela de Perguntas. Você verá que suas novas perguntas foram adicionadas na avaliação.

    Correspondente

    Para criar uma pergunta do tipo Correspondnte:

    1. Na tela de Perguntas, no menu drop-down Adicionar Pergunta:, selecione Correspondente.

    2. No campo "Valor do Ponto da Resposta", entre com o valor do ponto que você quer atribuir para esta pergunta. Entre com 0 (zero) para uma pergunta sem nota.

    3. No campo "Texto da Pergunta", entre com o texto para a pergunta. Clique em Exibir/Ocultar Editor Rich-Text para mostrar a barra de ferramentas WYSIWYG.

    4. Para adicionar um anexo, clique em Adicionar Anexos e entre com um nome ou procure um arquivo, ou entre com uma URL.

    5. Nos campos "Escolha" e "Correspondente", crie um par correto, depois clique em Salvar Emparelhamento para salvá-lo. Você pode fazer isso quantas vezes desejar, e você pode clicar em Editar ou Remover ao lado dos pares que você criou.

    6. No menu drop-down Atribuir por parte, escolha a parte (ou seja, seção) para a qual você deseja atribuir esta pergunta. Deixe-o com o valor padrão se você não criou nenhuma parte.

    7. Para atribuir esta pergunta para um conjunto de perguntas, no menu drop-down Atribuir a um conjunto de perguntas, selecione o nome do conjunto que você deseja.

    8. Sob "feedback de resposta Correta/Incorreta", você pode escolher fornecer um feedback de nível de pergunta.

    9. Clique em Salvar para voltar à tela de Perguntas. Você verá que suas novas perguntas foram adicionadas na avalização.

    Verdadeiro/Falso

    Para criar uma pergunta do tipo verdadeiro/falso:

    1. Na tela de Perguntas, no menu drop-down Adicionar Pergunta:, selecione Verdadeiro Falso.

    2. No campo "Valor do Ponto da Resposta", entre com o valor que você quer atribuir para esta pergunta. Entre com 0 (zero) para uma pertunta sem nota.

    3. No campo "Texto da Pergunta", entre com o texto para a pergunta. Clique em Exibir/Ocultar Editor Rich-Text para mostrar a barra de ferramentas WYSIWYG.

    4. Para adicionar um anexo, clique em Adicionar Anexos e entre com um nome ou procure um arquivo, ou entre com uma URL.

    5. No campo Resposta, selecione também Verdadeiro ou Falso.

    6. Sob "Requer Justificativa", você pode escolher para solicitar uma explicação clicando em Sim.

    7. No menu drop-down Atribuir por Parte, escolha a parte (ou seja, seção) para a qual você deseja atribuir esta pergunta. Deixe-o com o valor padrão se você não criou nenhuma parte.

    8. Para atribuir esta pergunta para um conjunto de perguntas, no menu drop-down Atribuir a um conjunto de perguntas, selecione o nome do conjunto que você deseja.

    9. Nos campos "resposta Correta (opcional)" e "resposta Incorreta (opcional)", você pode oferecer um feedback de nível de resposta.

    10. Quando terminar, clique em Salvar para voltar à tela de Perguntas. Você verá quer suas novas perguntas foram adicionadas na avaliação, juntamente com os botões para selecionar a resposta correta.

Gravação de Audio

Para criar uma pergunta do tipo gravação de audio:

  1. No menu drop-down, Adicionar Pergunta:, selecione Gravação de Audio.

  2. No campo "Valor do Ponto da Resposta", entre com o valor que você quer atribuir para esta pergunta. Entre com 0 (zero) para uma pertunta sem nota.

  3. No campo "Texto da Pergunta", entre com o texto para a pergunta. Clique em Exibir/Ocultar Editor Rich-Text para mostrar a barra de ferramentas WYSIWYG.

  4. Para adicionar um anexo, clique em Adicionar Anexos e entre com um nome ou procure um arquivo, ou entre com uma URL.

  5. Sob "Tempo permitido (segundos):" entre com o número de segundos que o aluno poderá utilizar para gravar a sua resposta.

  6. Sob Número de tentativas: escolha o número de vezes que o aluno pode regravar a resposta no menu drop-down.

  7. No menu drop-down Atribuir por Parte, escolha a parte (ou seja, seção) para a qual você deseja atribuir esta pergunta. Deixe-o com o valor padrão se você não criou nenhuma parte.
  8. Para atribuir esta pergunta para um conjunto de perguntas, no menu drop-down Atribuir a um conjunto de perguntas, selecione o nome do conjunto que você deseja.



  9. Sob "feedback de resposta Correta/Incorreta", você pode escolher fornecer um feedback de nível de pergunta.

  10. Clique em Salvar para voltar à tela de Perguntas. Você verá que suas novas perguntas foram adicionadas na avaliação.

Upload de Arquivo

Para criar uma pergunta do tipo upload de arquivo, a qual requer que o usuário envie um arquivo:

  1. Na tela de Perguntas, no menu drop-down Adicionar Pergunta:, selecione Upload de Arquivo.

  2. No campo "Valor do Ponto da Resposta", entre com o valor que você quer atribuir para esta pergunta. Entre com 0 (zero) para uma pertunta sem nota.

  3. No campo "Texto da Pergunta", entre com o texto para a pergunta. Clique em Exibir/Ocultar Editor Rich-Text para mostrar a barra de ferramentas WYSIWYG.

  4. Para adicionar um anexo, clique em Adicionar Anexos e entre com um nome ou procure um arquivo, ou entre com uma URL.

  5. Para atribuir esta pergunta para um conjunto de perguntas, no menu drop-down Atribuir a um conjunto de perguntas, selecione o nome do conjunto que você deseja.

  6. No campo "Feedback (opcional)", você pode oferecer um feedback de nível de pergunta.
  7. Quando terminar, clique em Salvar para voltar à tela de Perguntas. Você verá quer suas novas perguntas foram adicionadas na avaliação, juntamente com o ícone Upload de arquivo.

Copiar de um conjunto de Perguntas

Para copiar uma pergunta a partir de um conjunto de perguntas:

  1. Na tela de Perguntas, no menu drop-down Adicionar Pergunta: selecione Copiar de um conjunto de Perguntas.

  2. Clique no nome do conjunto do qual irá importar a pergunta.

  3. Navegue até o sob-conjunto e/ou use a(s) caixa(s) de texto para selecionar a(s) pertunga(s) que você deseja importar para a avaliação atual, depois clique em Copiar.
