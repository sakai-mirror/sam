Publicando uma avaliação

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

O ato de disponibilizar uma avaliação para seus usuários é apontado como uma publicação. Usuários não acessarão sua avaliação até que esta esteja publicada. Você deve fazer todas as modificações e correções para uma avaliação antes de publicá-la. Se você encontrar um erro ou desejar modificar de outra forma uma avaliação publicada, veja Modificando uma avaliação existente.

Para publicar uma avaliação que você criou:

  1. Na barra de menus do curso ou site de projeto, clique em Testes & Quizzes.

  2. Na página de Avaliações, sob "Avaliações Pendentes", clique no botão Configurações que corresponde à avaliação que você deseja publicar.

  3. Na página de Configurações, tenha certeza que todas as configurações foram feitas como você pretendia, e clique em Salvar Configurações e Publicar.

    Para mais informações sobre as configurações e suas opções, veja Configurações da Avaliação.

  4. Na página Confirmação de Publicação de Avaliação, você verá as configurações que você fez para a avaliação, mais uma URL que você pode usar para distribuir a avaliação. Clique em Publicar novamente para confirmar, ou Cancelar para abortar.

Na página de Avaliações, sua mais nova avaliação publicada aparecerá sob "Avaliações Publicadas"; uma cópia também permanecerá sob "Avaliações Pendentes" no caso de futuras modificações serem necessárias.
