Testes & Quizzes

Visão Geral

Esta ferramenta lhe permite criar avaliações online (ou seja, testes, quizzes e pesquisas) para disponibilizar via web para seus alunos ou outros grupos. Inicialmente foi projetada para administrar testes, mas você também pode criar avaliações para reunir informações de pesquisas ou feedback de curso. Mais...
