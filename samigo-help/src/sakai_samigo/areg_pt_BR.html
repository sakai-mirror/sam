Movendo perguntas entre as partes de uma avaliação

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

Para mover perguntas individuais de uma parte para outra dentro da avaliação:

  1. Na página de Avaliações, clique no título da avaliação que você deseja trabalhar. Você verá uma lista de perguntas para esta avaliação.

  2. Para cada pergunta que você deseja mover para uma parte diferente, clique em Editar.

  3. Na lista drop-down "Atribuir a uma Parte", selecione a parte onde você deseja que a pergunta apareça. Por exemplo, se você deseja que uma pergunta esteja na primeira parte, selecione 1.

  4. Clique em Salvar.

Para mover qualquer pergunta de uma parte para outra parte (ou serja, remover uma parte, mas manter suas pergutas):

  1. Clique em Remover próximo da parte que você está removendo.

    Nota: A primeira parte listada na tela de edição não terá a opção Remover. Para remover a primeira parte, você deve trocar sua ordem com outra parte (se você não tem outra parte, você terá que criar uma), e clicar em Remover.

  2. Selecione Remover somente a parte e mover a(s) pergunta(s) para, e use a lista drop-down que acompanha para escolher uma parte de destino.

  3. Click Remove.
