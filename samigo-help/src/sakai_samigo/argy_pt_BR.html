Tipos de perguntas disponíveis para uma avaliação

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

Você tem uma grande flexibilidade na criação de conteúdo para avaliações. Você pode usar cada tipo de pergunta listada abaixo alternadamente e em qualquer combinação dentro de uma avaliação. Para instruções sobre a criação de cada tipo de pergunta, veja Criando perguntas para uma avaliação.

O seguinte é uma visão geral sobre os tipos de perguntas disponíveis:

  • Multipla Escolha: Apresenta aos usuários uma pergunta seguida de uma lista de possíveis respostas. Você pode escolher para ter tanto uma quanto várias respostas corretas.

  • Pesquisa: Projetada para coleção de dados de pesquisa; Apresenta aos usuários uma pergunta ou declaração para a qual eles respondem usando uma escala de classificação da sua escolha:

    • Dictômica: Escolha "Sim", "Não" ou "Concordo", "Discordo".
    • Escalas Likert: Escolha entre os diferenciais semântico ou numérico (ou seja, Concordo plenamente - Discoro plemamente, 1-5)

  • Resposta curta/Resumo: Apresenta aos usuários uma pergunta seguida de uma caixa de texto na qual eles entram com a resposta

  • Preencha a lacuna: Apresenta aos usuários uma pergunta seguida de uma caixa de texto na qual eles entram com a resposta; cada resposta do usuário é comparada com uma lista de respostas permitidas

  • Rsposta numérica: Apresenta aos usuários uma pergunta seguida de uma caixa de texto na qual eles entram com uma resposta numérica; cada resposta do usuário é comparada com uma lista de respostas permitidas

  • Correspondente: Apresenta aos usuários uma série de escolhas e possíveis correspondências

  • Verdadeiro/Falso: Apresenta aos usuários uma declaração que eles devem identificar como sendo verdadeira ou falsa

  • Gravação de Audio: Apresenta aos usuários uma pergunta que eles devem responder de forma audível; um applet de gravação abre e permite o usuário gravar sua resposta usando um microfone

  • Upload de Arquivo: Apresenta uma pergunta ou atribuição que requer que o usuário faça o upload de um arquivo

  • Copiar de um Conjunto de Perguntas: Permite importar perguntas de um conjunto de perguntas existente que você tenha criado. Par mais informações, veja Adicionando, movendo, copiando ou removendo um conjunto ou subconjunto de perguntas.

Você pode definir um valor do ponto da resposta para cada pergunta. Perguntas de pesquisa são automaticamente definidas com 0 (zero) pontos, mas você pode (se você quiser) atribuir pontos para uma pergunta de pesquisa. Alem disso, cada uma das perguntas acima pontuada automaticamente, exceto para uma resposta curta/resumo, gravação de audio e perguntas de upload de arquivo.
