Usando o editor WYSIWYG

Em certas ferramentas, você pode controlar uma aparência gráfica para o seu texto usando o editor WYSIWYG. A barra de ferramentas WYSIWYG tem ícones para edição de estilização do seu texto.

Nota: Colando texto do Microsoft Word diretamente no editor WYSIWYG pode resultar em uma saida ilegível, sempre que o texto aparece corretamente no editor. Ao invés disso, para inserir texto formatado de um documento Word, clique no ícone Colar do Word, coloque o cursor na caixa de texto subsequente "Colar do Word", e tecle Ctrl-v no Windows, ou Cmd-v no Macintosh, para colar seu texto copidado. Usando Ctrl-v (or Cmd-v) para colar é o melhor método para preservação da formatação original do documento Word. Clique em OK para adicionar o texto para o item que você está compondo.

O editor WYSIWYG é paseado numa aplicação de código aberto (open-source application) chamada FCKeditor. Para ver o que cada ícone da barra de ferramentas faz, aponte para a página Barra de Ferramentas em FCKeditor Guia do Usuário.

Considerações sobre o Navegador

Para assegurar que o editor WYSIWYG apareça e funcione corretamente, use um navegador suportado; veja Qual navegador eu devo usar?

Entretanto, o Internet Explorer é atualmente o único que suporta a função colar dentro do editor. Se você está usando o Firefox, veja Problemas colando texto dentro do FCKeditor no Firefox.

O Internet Explorer também é atualmente o único navegador que suporta a característica de corretor automático. Para usar o corretor automática, você deve fazer primeiro o download e instalar a extensão ieSpell de Página de download do ieSpell; você somente pode fazer o download do ieSpell em um computador pessoal, não em uma estação de trabalho de uma universidade.
