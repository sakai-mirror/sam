Criando uma nova avaliação

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

  1. Na barra de menus do curso ou site de projeto, clique em Testes & Quizzes. Na página de Avaliações, você pode criar uma nova avaliação a partir do zero ou usando um template. (Para mais informações sobre o uso de templates, veja Adicionando, editando, ou deletando um Tipo de Avaliação.) Você pode ainda importar uma avaliação; veja Importando ou exportando uma avaliação.

    São quatro passos básicos para a criação de uma nova avaliação:

    • Definir o nome da sua avaliação
    • Definir as Configurações
    • Criar as partes
    • Criar as Perguntas

    Definir o nome da sua avaliação

    Na área da Nova AvaliaçãoIn, sob "Criar uma nova avaliação", digite um título para sua avaliação no campo "Título", depois clique em Criar. Isto leva à tela de edição.

    Alternativamente, clique em Criar Rápido, a qual lhe permite criar as perguntas da sua avaliação digitando ou copiando e colando adequadamente o texto formatado.

    Nota: As configurações padrão para sua avaliação será diferente dependendo se você clicar em Criar ou Criar Rápido; veja Diferenças entre Criar e Criar Rápido em Testes & Quizzes.

    Definir as Configurações

    O link Configurações, disponível na página de Avaliações ou na tela de edição, permite especificar vários parâmetros para a avaliação (ou seja, datas de entrega e configurações de segurança). Para mais informações sobre as possíveis configurações, veja Configurações da Avaliação.

    Nota: Você não deve usar as configurações de Feedback Imediato em Testes & Quizzes quando se avalia o aprendizado, pois pode fornecer informações que o aluno pode usar para corrigir suas respostas antes de submeter um teste. Feedback Imediato é uma configuração padrão quando você clica em Criar (e não em Criar Rápido) para começar sua avaliação, sem usar um Tipo de Avaliação com uma configuração padrão diferente.

    Criar partes

    Quando você cria uma nova avaliação, uma parte (ou seja, seção) chamada "Padrão" é criada automaticamente. Se você deixá-la com o nome "Padrão", o título não aparecerá na sua avaliação; para alterar o nome da parte, clique em Editar. Você pode começar adicionando perguntas imediatamente para "Padrão", ou pode adicionar suas próprias partes.

    Para adicionar uma parte para sua avaliação:

    1. Na tela de edição, clique em Adicionar Parte.

    2. Na tela de edição, próximo de "Título", digite o nome para esta parte.

    3. Sob "Informação", você pode usar o editor WYSIWYG para entrar com uma breve descrição ou instruções.

    4. Se você quiser adicionar anexos, clique em Adicionar Anexos.

    5. Sob "Tipo", marque a opção para indicar se você criará as perguntas uma a uma, ou usará perguntas aleatoriamente do seu conjunto de perguntas. Como última opção, use a lista drop-down para escolher o nome do conjunto do qual as perguntas serão utilizadas e use as caixas de texto que a companham para indicar o número e valor do ponto das perguntas. Marque as opções para indicar o tipo de ordenação aleatória.

    6. Sob "Ordenação de Perguntas", marque as opções para ordenar as perguntas ou permitir uma ordenação aleatória dentro da parte.

    7. Sob "Metadados", você pode usar as caixas de texto para gravar qualquer objetivo, palavras-chave, ou abreviações.

    8. Para salvar suas alterações, clique em Salvar. Para cancelar, clique em Cancelar.

    As novas partes serão listadas na ordem que você criá-las. Para mudar a ordem de duas partes, antes do nome da parte, altere o número na lista drop-down (ou seja, próximo de "Parte"). Por exemplo, se você tem três partes, e você deseja que a terceira parte apareça primeiro, use a lista drop-down para mudar o 3 para 1. A terceira parte ficará em primeiro e a primeira ficará em terceiro na ordem.

    Para editar o nome e qualquer das informações e configurações associadas com uma parte, próximo do nome da parte, clique em Editar.

    Sua avaliação sempre deve conter pelo menos uma parte, mas você pode remover qualquer uma das partes que você criar. Você pode também remover a parte "Padrão", contanto que você já tenha criado outra parte para substituí-la.

    Para remover uma parte:

    Nota: A primeira parte listada na tela de edição não terá a opção Remover. Para remover a primeira parte, você deve trocar sua ordem com outra parte (if você não tem outra parte, você terá que criar uma), então clicar em Remover.

    1. Próximo do nome da parte, clique em Remover.

    2. Na tela de confirmação subsequente, escolha entre o seguinte:

      • Remover a parte e todas as perguntas
      • Remover somente a parte e mover a(s) pergunta(s) para (use a lista drop-down que acompanha para escolher outra parte)

    3. Clique em Remover.

    Criar perguntas

    Você pode também criar uma nova pergunta para adicionar para a avaliação, ou você pode adicionar uma pergunta de um conjunto de perguntas. Para adicionar sua primeira pergunta, crie antes uma avaliação, e então no menu drop-down próximo de "Adicionar Pergunta:", escolha o tipo de pergunta para adicionar. para mais informações em tipos de pergunta, veja Criando perguntas para uma avaliação. Uma vez que você tenha selecionado um tipo de pergunta, a tela de edição de pergunta se abrirá.

    Para adicionar uma pergunta de um conjunto de perguntas, no menu drop-down próximo de "Adicionar Pergunta", selecione Copiar de um conjunto de Perguntas.

    Na tela de edição de perguntas, as opções disponíveis dependerão de qual tipo de pergunta você está adicionando. Para mudar o tipo da pergunta, escolha outra opção do menu drop-down "Mudar Tipo de Pergunta" drop-down. Opções comuns para a maioria das perguntas incluem:

    • Valor do Ponto da Resposta: Especifique o valor do ponto para cada resposta numa pergunta, ou especifique 0 (zero) para uma pesquisa ou pertunga sem nota.

      Nota: O total de pontos que você atribuir será dividido pelo total de lacunas da pergunta. Por exemplo, pegue a seguinte pergunta:

      Polonius avisa Laertes: "Seja um {credor} e não um {devedor}".

      Se você atribuir 10 pontos para esta pergunta, o aluno obterá 5 pontos para cada lacuna "credor" and "devedor".

    • Texto da Pergunta: Informe o sentido real da pergunta a ser feita.

    • Resposta: Indique a resposta ou respostas corretas. Você tem a opção para incluir feedback para respostas específicas (feedback a nível de item).

    • Atribuir a uma parte: Se você criou suas próprias partes, você pode atribuir uma pertunta para qualquer uma destas partes; caso contrário, será atribuida à parte padrão.

    • Atribuir para um Conjunto de Perguntas: Adicione esta pergunta para um conjunto de perguntas que você criou. Para mais informações sobre conjunto de perguntas, veja Adicionando, movendo, copiando, ou removendo um conjunto ou subconjunto de perguntas.

    • Feedback (opcional): Fornece feedback para uma resposta correta ou incorreta da pergunta (feedback a nível de pergunta).

    • Salvar: Clique em Salvar para adicionar a pergunta para sua avaliação ou para salvar qualquer alteração da pergunta.

    • Cancelar: Clique em Cancelar para abortar a pergunta ou alterações e voltar à tela de edição.

    Após inserir uma pergunta na sua avaliação, uma lista drop-down, Inserir Nova Pergunta, surgirá na tela de edição de avaliação, no final de cada parte. User esta opção para adicionar uma pergunta diretamente para uma parte específica. Próximo do topo, na barra de título da avaliação, clicando em Visualizar Avaliação lhe permite visualizar a avaliação como está no momento. Depois que você salvou as perguntas dentro da sua avaliação, você pode usar os botões correspondentes Editar e Remover para alterar ou deletar perguntas.
