Criando uma pesquisa

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

Você pode criar uma pesquisa usando o mesmo processo que você usaria para criar uma avaliação, desde que o tipo de pergunta da "Pesquisa" permita-o escolher de um conjunto predefinido de escalas comumente utilizado para relatório de pesquisa. Tais escalas são as seguintes:

  • Sim, Não
  • Concordo, Discordo
  • Concordo, Indeciso, Discordo
  • Abaixo da Média -> Acima da Média
  • Concordo Plenamente -> Discordo Plenamente
  • Excelente -> Inaceitável
  • 1 -> 5
  • 1 -> 10

Por isso, para criar uma pesquisa, simplesmente crie uma avaliação que consiste das perguntas da "Pesquisa". Para mais informações, veja Criando perguntas para uma avaliação.
