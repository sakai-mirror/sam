Sobre Tipos da Avaliação

Um Tipo da Avaliação é um conjunto pré-selecionado de ajustes de avaliação que pode tornar mais simples para você criar avaliações. Tipos de Avaliação são compartilhados automaticamente através do seu curso e projetos de sites.

Quando você cria ou edita um Tipo de Avaliação, você determina que alterações de ajustes você pode fazer quando você usa o Tipo de Avaliação para criar avaliações. Para mais informações, veja Criando uma nova avaliação e Configuração da avaliação.

Criando um Tipo de Avaliação

Para criar um Tipo de Avaliação:

  1. Na barra de menus, clique em Testes & Quizzes.

  2. Na página de Avaliações, clique em Tipos de Avaliação.

  3. Sob "Novo Tipo de Avaliação", no campo "Título", digite o nome para seu Tipo de Avaliação, e depois clique em Criar.

  4. Sob "Editar Tipo de Avaliação", você verá uma lista de configurações. Para mostrar as opções para uma configuração, clique no seu nome ou no ícone de triângulo ao lado. Então, marque as caixas para selecionar as configurações e opções que você pode editar. Alguns ajustes também têm valores padrão que você pode configurar. Para mais informações sobre configurações e opções disponíveis, veja a seção abaixo Configurações do Tipo de Avaliação .

  5. Quando terminar, clique em Salvar. Você verá seus Tipos de Avaliação listados sob "Tipos de avaliação salvos".

Editando um Tipo de Avaliação

Para editar uma Tipo de Avaliação, na tela Tipos de Avaliação, sob "Tipos de avaliação salvos", clique no nome do Tipo de avaliação. Para mais informações sobre configurações e opções, veja a seção abaixo Configurações de Tipos de Avaliação. Quando terminar, clique em Salvar.

Deletando Tipo de Avaliação

Para deletar um Tipo de Avaliação, na tela Tipos de Avaliação, sob "Tipos de avaliação salvos", clique no link Remover abaixo de Tipo de Avaliação para ser removido. Clique em Remover novamente para confirmar.

Configurações do Tipo de Avaliação

Quando você cria ou edita um Tipo de Avaliação, você determina que alterações de ajustes você pode fazer quando cria uma avaliação baseada no Tipo de Avaliação. As seguintes configurações e opções estão disponíveis:

  • Informação do Tipo de Avaliação: Nos campos fornecidos, Defina o título, Autor e descrição para seu Tipo de Avaliação. Para fazer esta informação aparecer nas configurações para todas as avaliações que usa este Tipo de Avaliação, marque a caisa ao lado Pode ser visualizado? .

  • Introdução da Avaliação: Marque estas caixas para tornar possível editar os campos "Autor" e "Descrição".

  • Avaliação liberada para: Use esta opção para determinar quem pode tomar uma avaliação que está baseada neste Tipo de Avaliação.

  • Segurança Alta: Marque estas caisas para tornar possível definir restrições de segurança opcional nas avaliações. Se você deixar estas caixas desmarcadas, as configurações de "Alta Segurança"the "High Security" estarão indisponíveis em avaliações criadas com esse tipo de Avaliação.

  • Avaliação Cronometrada: Se você marcar esta caixa, você terá a opção que torna a availação cronometrada e para auto-submeter as respostas se o tempo se esgotar. Se você deixar esta caixa desmarcada, as configuranções da "Avaliação Cronometrada" estarão indisponíveis nas avaliações criadas com esse Tipo de Avaliaçao.

  • Organização da Avaliação: Estas configurações controlam o fluxo da avaliação como os alunos a tomam. Marque as opções para escolher os valores padrão para estas configurações; marque as caixas para permitir alterações. Se você deixar estas caixas desmarcadas, os valores padrão sempre terão efeito nas avaliações criadas com este Tipo de Avaliação.

  • Submissões: Estas configurações controlam o tempo no qual os alunos podem tomar a avaliação, e se eles podem ou não tomar a avaliação depois da data de vencimento. Marque as opções para escolher os valores padrão para estas configurações; marque as caixas para permitir alterações. Se você deixar estas caixas desmarcadas, os valores padrão sempre terão efeito nas avaliações criadas com este Tipo de Avaliação.

  • Mensagem de Submissão: As Mensagens de Submissão e a URL da página de destino aparece na última página da avaliação, depois do aluno clicar em Submeter para Classificação. Marque estas caixas para permitir a inserção de mensagem personalizadas e URLs aqui. Se você deixar estas caixas desmarcadas, estas opções estarão indisponíveis em todas as avaliações criadas com esse Tipo de Avaliação.

  • Feedback: Sob "Criação de Feedback", você pode escolher Feedback a Nível de Pergunta, Feedback a Nível de Seleção, ou Ambos. Sob "Entrega de Feedback", você pode escolher se ou quando os alunos poderão ver o feedback. Sob "Componentes de Feedback que os Alunos Podem Ver", você escolher que tipos de feedback eles podem ver. Você pode também escolher se você poderá ou não alterar estas configurações.

    Nota: Você não deveria usar as configurações de Feedback Imediato em Testes & Quizzes quando se avalia o aprendizado, pois pode fornecer informações que os alunos podem usar para corrigir suas respostas antes de submeter uma teste. O Feedback Imediato é uma configuração padrão quando você clica em Criar (e não em Criar Rápido) para começar sua avaliação, sem usar uma Tipo de Avaliação com uma configuração padrão diferente.

  • Classificação: A opção Classificação controla o seguinte:

    • Se os Classificadore pode ver os nomes dos alunos ou se a classificação é anônima
    • Se as notas das avaliações são ou não enviadas para a ferramenta Quadro de Notas
    • Como as notas múltiplas para múltiplas submissões são manipuladas

    Marque as opções para escolher o valor padrão para estas configuranções; marque as caixas para determinar se você pode ou não alterá-las. Se você deixar as caixas desmarcadas, os valores padrão sempre terão efeito nas avaliações criadas com este Tipo de Avaliação.

  • Gráficos: Marque esta caixa para permitir alterações na cor de fundo ou imagens de fundo das avaliações. Se você deixar esta caixa desmarcada, as opções "Gráficas" estarão indisponíveis em todas avaliações criadas com esse Tipo de Availação.

  • Metadados: As configurações de "Metadados" são campos de texto para gravação de informações úteis sobre uma avaliação ou uma pergunta. Os Três campos de metadados disponíveis são "Objectivo", "Palavra-chave" e "Rubrica". Se Você deixar estas caixas desmarcadas, os metadados não serã gravados nas avaliações criadas com esse Tipo de Avaliação.
